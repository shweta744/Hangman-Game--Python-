•Hangman Game
•A simple Python-based Hangman game created as part of my programming practice and internship task.
Description
This project is a text-based Hangman game developed using Python. The computer randomly selects a word, and the player has to guess it letter by letter within a limited number of attempts.
•Features
Random word selection
Limited wrong guesses
User-friendly interface
Displays guessed letters
Win and lose conditions
•Technologies Used
Python
•How to Run
Download the hangman.py file.
Open it in any Python IDE or compiler.
Run the program.
Start guessing the letters and enjoy the game!
